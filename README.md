# ♟️ Movimentação de Peças de Xadrez em C

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como parte de um desafio acadêmico com o objetivo de simular a movimentação de peças de xadrez utilizando a linguagem C.

O foco principal foi aplicar conceitos fundamentais e avançados de programação, como recursividade, loops aninhados e controle de fluxo, simulando o comportamento das peças: Rainha, Torre, Bispo e Cavalo.

---

## 🎯 Objetivo

Implementar a lógica de movimentação das peças de xadrez no console, respeitando regras específicas definidas no desafio, e demonstrando domínio das estruturas de repetição e recursividade.

---

## ⚙️ Tecnologias Utilizadas

- Linguagem C
- Compilador GCC
- VS Code

---

## 🧠 Conceitos Aplicados

Durante o desenvolvimento, foram utilizados os seguintes conceitos:

- 🔁 Recursividade (Rainha, Torre e Bispo)
- 🔄 Loops (for, while)
- 🔂 Loops aninhados (Bispo e Cavalo)
- ⛔ Controle de fluxo com `break`
- 🧩 Estruturas de decisão (`switch`)
- 🧾 Boas práticas de organização e indentação
- 📝 Documentação de código

---

## ♟️ Movimentação das Peças

### ♛ Rainha
- Movimenta-se verticalmente utilizando recursividade

### ♜ Torre
- Movimenta-se horizontalmente utilizando recursividade

### ♝ Bispo
- Movimento diagonal utilizando recursividade + loops aninhados

### ♞ Cavalo
- Movimento em "L" (duas casas para cima e uma para a direita)
- Implementado com loops e controle de fluxo (`break`)
